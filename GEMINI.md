# vw_convertor (px to vw 자동 변환 도구)

이 프로젝트는 CSS 파일 내의 `px` 단위를 특정 기준 너비(Base Width)를 바탕으로 `vw` 단위로 일괄 변환해주는 Node.js 기반의 유틸리티 도구입니다.

## 🚀 프로젝트 개요
- **목적**: 반응형 및 모바일 웹 개발 시, 디자인 가이드의 `px` 수치를 일일이 `vw`로 계산하는 번거로움을 해결합니다.
- **주요 기술**: Node.js (File System API)
- **핵심 로직**: `(px 값 / 기준 너비) * 100` 공식을 사용하여 소수점 4자리까지 정밀하게 변환합니다.

## 🛠 사용 방법

### 1. 설정 확인 및 변경
`convert_to_vw.js` 파일 상단의 `설정 부분`에서 대상 파일과 기준 너비를 지정합니다.
```javascript
// convert_to_vw.js
const TARGET_FILE = 'convert_to_vw.css'; // 변환할 CSS 파일명
const BASE_WIDTH = 414;                // 디자인 가이드 기준 너비 (예: iPhone 11)
```

### 2. 스크립트 실행
터미널에서 아래 명령어를 입력하여 변환을 수행합니다.
```bash
node convert_to_vw.js
```

## 📂 주요 파일 설명
- **`convert_to_vw.js`**: 변환 로직이 담긴 메인 스크립트입니다. 정규표현식을 사용하여 CSS 내의 `px` 패턴을 찾아 변환합니다.
- **`convert_to_vw.css`**: 변환 대상이 되는 CSS 파일입니다. (설정에서 파일명 변경 가능)

## ⚠️ 주의 사항
- **직접 수정(Overwrite)**: 스크립트 실행 시 원본 파일이 즉시 변경되므로, 실행 전 반드시 백업을 권장합니다.
- **0px 유지**: `0px`은 변환하지 않고 그대로 유지합니다.
- **중복 변환 방지**: 이미 `vw`로 변환된 수치는 건드리지 않으며, 새로 추가되거나 수정된 `px` 수치만 찾아 변환합니다.

## 💡 개발자 팁 (10년차 프론트엔드 관점)
- 모바일 전용 스타일시트 작업 시 매우 유용합니다.
- `BASE_WIDTH`는 프로젝트 시작 시 디자이너가 전달한 시안의 가로 너비(예: 360, 375, 414 등)와 반드시 일치시켜야 정확한 결과가 나옵니다.
- 소수점 처리가 `toFixed(4)`로 되어 있어 브라우저 렌더링 오차를 최소화합니다.
